Getting Started

You’ll start by learning about two complementary things: python, and the ideas behind machine learning. It seems to be easiest to do both of these at the same time. Introduce yourself to python with an online course. Here are some recommendations in no particular order:

  • Codecademy
  • DataCamp
  • Learn Python the Hard Way ($30 up front)

Once you’ve seen a little python, start on an introductory kaggle project.

  1. Titanic: Machine Learning from Disaster - includes tutorial - start here!
  2. Digit Recognizer - computer vision & neural networks with classic MNIST data set

When you’re ready to install python on your computer, you have two options:

  1. Download anaconda (python version 3.6), which includes python, scikit-learn, jupyter notebook, and a zillion other things you’ll use with python.
  2. If you like to keep a closer eye on what’s installed on your computer, download python directly (version 3.6). Then you can download packages like numpy and scipy separately as you need them.
